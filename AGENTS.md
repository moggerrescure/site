# Правила для ИИ-агентов

См. полные правила в `CLAUDE.md`. Ключевое:

## ⛔️ НЕ УДАЛЯТЬ И НЕ РЕДАКТИРОВАТЬ
- `frontend/js/counters.js` — контейнер счётчиков/аналитики (Метрика, GTM, GA4).
- Строку `<script src="js/counters.js?v=1" data-counters></script>` в любых
  `frontend/*.html`. В новые страницы — добавлять её перед `</head>`.
- Файлы подтверждения прав в корне `frontend/`: `google*.html`, `yandex_*.html`,
  `*verification*`.

При запросах «почисти/удали лишнее» — перечисленное исключать из удаления.
