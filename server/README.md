# 🕯️ Memory Site — Backend Server

Node.js HTTP server с SQLite базой данных.  
**Без внешних зависимостей** — работает на чистом Node.js 24+.

---

## 🚀 Запуск

```bash
# 1. Перейти в папку сервера
cd server

# 2. Скопировать .env
cp .env.example .env
# Отредактировать .env: задать JWT_SECRET

# 3. Запустить
node --experimental-sqlite index.js
```

Сервер запустится на **http://localhost:3000**  
Фронтенд открывается там же автоматически.

---

## ⚙️ Конфигурация (.env)

| Переменная    | Описание                          | По умолчанию |
|---------------|-----------------------------------|--------------|
| `PORT`        | Порт сервера                      | `3000`       |
| `JWT_SECRET`  | Секрет для подписи JWT токенов    | (обязательно в проде) |

## 📡 API Endpoints

### Люди
| Метод  | URL                        | Описание                        | Auth |
|--------|----------------------------|---------------------------------|------|
| GET    | `/api/people`              | Список (page, limit, q, city)   | —    |
| GET    | `/api/people/:id`          | Один человек + его отзывы       | —    |
| POST   | `/api/people`              | Создать                         | ✅   |
| PUT    | `/api/people/:id`          | Обновить                        | ✅   |
| DELETE | `/api/people/:id`          | Удалить                         | ✅   |
| POST   | `/api/people/:id/photo`    | Загрузить фото (multipart)      | ✅   |

### Отзывы
| Метод  | URL                           | Описание                | Auth |
|--------|-------------------------------|-------------------------|------|
| GET    | `/api/reviews/:personId`      | Все отзывы к человеку   | —    |
| POST   | `/api/reviews/:personId`      | Оставить отзыв          | —    |
| DELETE | `/api/reviews/delete/:id`     | Удалить отзыв           | ✅   |

### Свечи
| Метод | URL                  | Описание               |
|-------|----------------------|------------------------|
| GET   | `/api/candles`       | Текущий счётчик        |
| POST  | `/api/candles/light` | Увеличить счётчик на 1 |

### Авторизация
| Метод | URL                    | Описание           |
|-------|------------------------|--------------------|
| POST  | `/api/auth/register`   | Регистрация        |
| POST  | `/api/auth/login`      | Вход, возвращает JWT |
| GET   | `/api/auth/me`         | Текущий пользователь |

### Статистика
| Метод | URL          | Описание                               |
|-------|--------------|----------------------------------------|
| GET   | `/api/stats` | people, reviews, candles, cities count |

## 📂 Структура

```
server/
├── index.js      — точка входа, HTTP сервер + статика
├── router.js     — все роуты и handlers
├── db.js         — SQLite схема и инициализация
├── auth.js       — JWT (HS256) + PBKDF2 хеширование паролей
├── upload.js     — multipart/form-data парсер для фото
├── .env          — конфигурация (не в git)
├── .env.example  — пример конфигурации
├── data/         — SQLite файл (не в git)
└── uploads/      — загруженные фото (не в git)
```

---

## 🔐 Авторизация

Используется **JWT (HS256)** без зависимостей.  
Пароли хранятся в виде `PBKDF2-SHA512` с солью (100 000 итераций).

Для защищённых запросов передавайте заголовок:
```
Authorization: Bearer <token>
```

---

## 🗄️ База данных

SQLite через встроенный модуль `node:sqlite` (Node 24+).  
База создаётся автоматически в `server/data/memory.db`.  
При первом запуске засевается **18 людьми и отзывами**.
